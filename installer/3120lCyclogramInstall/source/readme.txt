v0.5
 - исправлено падение при переходе в последовательности Stop>Pause>Play (переход в паузу подразумевал паузу текущего режима, в стопе он становился Nil)
 - исправлено отображение текущего режима при переходе из Stop в Play (отрисовка зеленым текущего режима происходит по условию что на момент
   отрисовки режим active. Отрисовка происходила по кнопке Play, но active режима взводится только на следующей итерации Recorder)
 - отображение состояния циклограммы разбито на 2 stringgrid - с  режимами и с измерениями (чтобы измерения всегда
   были на экране независимо от количества режимов). StringGrid с измерениями синхронизирован по строкам с основным, чтобы при скролинге строк основного
   строка измерений первого контрола соответствовала первой строке с режимами первого контрола
 - При остановке циклограммы больше не устанавливаются задания в 0 всем контролам. Для CMNControl устанавливается в 1 команда cmdStop,
 При старте она устанавливается в 0 и потом перезаписывается значением, которое предустановлено для режима.
 - На красную кнопку выведена форма настройки расчета передач. 
 - Добавлен переход на режим при срабатывании передачи трансмиссии. Переход сработает если есть соответствующая галочка на форме.
 - Если конфиг отсутствует идет настройка по программе испытаний.
 - Уставки задаются абсолютными значениями. Normal уровень если 0 то допуски означают абсолютные уровни. Если Normal<>0 то Уставка=Normal+HH,h,l,LL
 - Уставки L и LL отключены циклограммой
