# Заметки по реализации динамических диалогов настроек компонентов

1. Создан модуль uComponentSettingsDialog.pas в кодировке cp1251, содержащий TForm наследника для гибкой настройки.
2. Реализована интерактивная фильтрация списка тегов в ComboBox по мере ввода в TEdit подстроки.
3. Поддерживается сохранение шрифта (вызывается стандартный TFontDialog) и сохранение/загрузка состояния в gui.ini.
4. Подсветка тревог в sgFormular реализована через Canvas.Brush.Color в событии OnPrepareCanvas.
