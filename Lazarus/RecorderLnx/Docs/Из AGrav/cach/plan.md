## Design-time режим редактора мнемосхем

- Кнопка `\` на тулбаре полотна стала переключателем режима редактирования.
- В режиме редактирования можно выбирать компоненты яркой рамкой; `Ctrl` добавляет/снимает компонент из множественного выбора.
- Протяжка рамки по пустому полотну выбирает все компоненты с частичным пересечением.
- Выбранные компоненты можно перетаскивать мышью; общий красный контейнер с ручками масштабирует всю выбранную группу.
- Клавиатура: стрелки двигают выбранные компоненты на 1 пиксель; `Ctrl+стрелки` меняют ширину/высоту на 1 пиксель; `Ctrl+Shift+стрелки` меняют ширину/высоту на 5 пикселей.
- Проверено: основной проект собран во временный `RecorderLnx_check.exe`; `RecorderFormModelTest.exe` прошёл.

## Вынос редактора мнемосхем из TMainForm

- Код design-time редактирования вынесен из `uMainForm.pas` в новый модуль `UI\uFormEditorController.pas`.
- `TFormEditorController` подключается к LCL-полотну, вешает обработчики мыши/клавиатуры и работает через callback активной `TRecorderFormPage`.
- Редактор отключается через `Enabled := False`, когда кнопка `\` отжата: выбор очищается, активная операция завершается, edit-функции больше не обрабатываются.
- Прямые WinAPI/window messages не используются; для Windows/Linux переносимости применены кроссплатформенные LCL-события (`OnMouseDown`, `OnMouseMove`, `OnMouseUp`, `HandleKeyDown`).
- `uMainForm.pas` оставлен UI shell: создает контроллер, передает активную страницу, добавляет тестовые компоненты и вызывает render.
- Проверено: основной проект собран во временный `RecorderLnx_check.exe`; `RecorderFormModelTest.exe` прошёл.

## Cursor feedback редактора мнемосхем

- `TFormEditorController` обновляет курсор при наведении в режиме редактирования: над телом компонента ставится `crSizeAll`, над resize-ручками и горячими зонами границ выбранного компонента ставятся resize-курсоры.
