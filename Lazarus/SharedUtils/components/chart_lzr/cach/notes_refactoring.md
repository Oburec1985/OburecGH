# Заметки по рефакторингу архитектуры слушателей (Chart Listener Architecture)

## Проблема
Модуль `uOglChartFrameListener.pas` разросся до ~2250 строк кода. Он содержал как базовый класс `TChartFrameListener`, так и сложные, не связанные друг с другом логики:
1. Выделение страниц и осей.
2. Панорамирование и зум холста, а также функции FitZoom.
3. Интерактивное изменение геометрии страниц (ресайз границ, перемещение через Shift, примагничивание).
4. Редактирование вершин Безье трендов (перетаскивание вершин, усов, клавиши добавления/удаления/смены типа точек).
5. Редактирование текстовых меток и флагов (перетаскивание, ресайз меток, авто-сортировка наложений флагов).

Это затрудняло отладку, расширение и тестирование отдельных функций.

## Решение и архитектура
Модуль был декомпозирован на 6 компактных специализированных модулей:
1. **`uOglChartFrameListener.pas`**
   - Базовый абстрактный класс `TChartFrameListener`.
   - Общая процедура логирования событий `LogToFile`.
2. **`uOglChartSelectListener.pas`**
   - Класс `TChartSelectListener`.
   - Выделение страниц и осей кликом мыши.
   - Подсветка `HoveredObject` при наведении и изменение формы курсора над осями.
3. **`uOglChartPanZoomListener.pas`**
   - Класс `TChartPanZoomListener`.
   - Масштабирование графика колесиком мыши.
   - Панорамирование (перетаскивание графиков правой кнопкой мыши).
   - Выделение прямоугольной рамки для приближения (Ctrl + левая кнопка мыши).
   - Глобальные процедуры автоподгонки масштаба `FitZoomX`, `FitZoomY`, `FitZoomXY`, `FitPageZoom`.
4. **`uOglChartPageGeometryListener.pas`**
   - Класс `TChartPageGeometryListener`.
   - Изменение размеров страниц мышью за левую/правую/верхнюю/нижнюю границы.
   - Перемещение страниц по холсту с зажатой клавишей Shift.
   - Примагничивание (Snap) границ страниц друг к другу с настраиваемой чувствительностью `SnapSensitivity`.
5. **`uOglChartVertexEditListener.pas`**
   - Класс `TChartVertexEditListener`.
   - Выделение вершин Безье трендов (кликом по вершине).
   - Перетаскивание вершин трендов и их направляющих усов (`cdpLeft` и `cdpRight` для сглаженных точек).
   - Обработка горячих клавиш: `DELETE` (удаление точки), `INSERT` (добавление точки в позицию под курсором), `1`/`2`/`3`/`T` (изменение типа точки на угловую/сглаженную/скрытую).
6. **`uOglChartLabelEditListener.pas`**
   - Класс `TChartLabelEditListener`.
   - Выделение, перемещение и ресайз текстовых меток за границы и углы.
   - Смена курсора мыши при наведении на рамки меток (`crSizeWE`, `crSizeNS`, `crSizeNWSE`, `crSizeAll`).
   - Математические функции расчета пиксельных размеров меток `GetTextLabelPixelRect` и алгоритм расталкивания флагов по вертикали `SortFlagsOnPage`.

## Регистрация слушателей
Все слушатели автоматически регистрируются в конструкторе `TOglChart.Create` в файле `uoglchart.pas`:
```pascal
  AddFrameListener(TChartPanZoomListener.Create);
  AddFrameListener(TChartPageGeometryListener.Create);
  AddFrameListener(TChartVertexEditListener.Create);
  AddFrameListener(TChartLabelEditListener.Create);
  AddFrameListener(TChartSelectListener.Create);
```

## Стабильность пакета
Все новые файлы зарегистрированы в Lazarus-пакете `lzrObrPack.lpk`. 
Все файлы сохранены в кодировке `Windows-1251` (`cp1251`), без `BOM`, с переводами строк `CRLF`. Русские комментарии бережно перенесены и структурированы.
