# Заметки по реализации интерактивного курсора (TChartCursor)

## Выполненные задачи:
1. Создан модуль uOglChartCursor.pas с определением класса TChartCursor, поддерживающим типы cctSingle и cctDouble, свойства X1, X2, цвета, смещения меток и сериализации/десериализации JSON.
2. Модуль uOglChartSerializer.pas дополнен веткой для десериализации TChartCursor.
3. Рендерер (uOglChartRenderer.pas) дополнен методом DrawCursor, который:
   - Рисует пунктирную линию для одинарного и двойного курсора.
   - Отрисовывает светлые полупрозрачные текстовые метки (DrawCursorLabel) с фоном , обводкой цветом курсора и темным текстом для высокой читаемости.
   - Метка выровнена по левой границе (левая граница флага прилеплена к линии курсора с отступом 2 пикселя).
   - Вычисляет значения выбранного тренда (FindSelectedTrend, GetTrendValueAtX) с линейной интерполяцией.
   - Поддерживает расчет координат как для линейной, так и для логарифмической оси X (исправлен метод PixelToXValue).
4. Создан слушатель событий uOglChartCursorListener.pas, который:
   - Обрабатывает захват/перемещение линий курсора и текстовых меток.
   - Реализует притягивание к экстремумам выбранного тренда с зажатым Shift в радиусе 5 пикселей (метод GetSnappedX).
   - Обрабатывает хоткей Alt+3 для переключения видимости и режимов курсора.
   - Обрабатывает Home / Ctrl+Home для мгновенного перемещения линий курсора.
   - Управляет формой курсора мыши (crSizeWE для линий, crSizeNS для меток).
5. Приоритет слушателей изменен в uoglchart.pas: TChartCursorListener добавлен первым в список fListeners, что позволило перехватывать события мыши до того, как они будут поглощены выбором страниц или зумом.
6. Написан тест uOglChartCursorTest.pas для проверки корректности работы курсора и демонстрации его использования.
7. Создан автоматизированный скрипт test_cursor_interactive.py, который успешно протестировал включение курсора, перетаскивание линии курсора и перетаскивание метки, сохранив скриншоты.

## Текущее состояние:
- Все требования задачи измерительного курсора выполнены на 100%.
- Тесты успешно собираются, интерактивное перетаскивание работает превосходно.
