# Заметки по реализации системы фрейм-листенеров (Frame Listeners) в OGlChart

## Цель доработки
Добавить гибкую, расширяемую систему взаимодействия с графиком (выбор объектов, подсветка при наведении, перетаскивание диапазона осей, масштабирование колесиком мыши относительно курсора) по аналогии со слушателями кадров (Frame Listeners) в 3D-движках.

## Архитектура системы

1. **Интерфейс `IChartControl` (`uOglChartTypes.pas`)**
   Используется для предоставления слушателям доступа к управляющему элементу (`TOglChart`) без прямой зависимости от класса (для избежания циклических ссылок между модулями):
   - `GetRenderer: TObject` — возвращает ссылку на экземпляр `TOpenGLChartRenderer`.
   - `GetModel: TObject` — возвращает ссылку на модель `TChartModel`.
   - `Redraw` — вызывает перерисовку графика.

2. **Базовый класс `TChartFrameListener` (`uOglChartFrameListener.pas`)**
   Определяет виртуальные методы обработки событий:
   - `FrameStarted`, `FrameEnded` — вызываются до и после отрисовки кадра.
   - `MouseDown`, `MouseMove`, `MouseUp`, `MouseWheel` — события мыши.
   - `KeyDown`, `KeyPress` — события клавиатуры.

3. **Слушатели по умолчанию**
   - **`TChartSelectListener`**:
     - При наведении мыши определяет объект под курсором (сначала текстовые метки осей через `fTextHits` рендера, затем страницы по их границам) и устанавливает `HoveredObject` в рендере.
     - При клике левой кнопкой мыши устанавливает `SelectedObject` в рендере.
   - **`TChartPanZoomListener`**:
     - Обрабатывает перетаскивание правой кнопкой мыши (pan) для прокрутки осей X страницы и Y графиков.
     - Обрабатывает колесико мыши (zoom) для масштабирования графика относительно текущего положения курсора мыши.

4. **Подсветка объектов (`uOglChartRenderer.pas`)**
   - В метод `Render` добавлен вызов `DrawHighlights`.
   - Если задан `HoveredObject` (страница или ось), рендер рисует полупрозрачную желтую рамку вокруг границ объекта.
   - Если задан `SelectedObject`, рендер рисует жирную оранжево-красную рамку.

5. **Тест и пример использования (`Test_component/unit1.pas`)**
   - Добавлен класс `TTestLogListener`, который переопределяет метод `MouseDown` и выводит имя выбранного и наведенного объекта в консоль/лог при каждом клике.
   - Данный слушатель автоматически регистрируется в `FormCreate`.

## Новые возможности интерактивного взаимодействия (Редактирование страниц и текста)

1. **Редактирование текста при двойном клике**:
   - При двойном клике (ssDouble in Shift) на текстовом поле осей/страниц выбирается вся строка (EditSelectionStart = 1, EditSelectionLength = Length(fEditText)).
   - При вводе первого символа вся строка очищается и заменяется новым вводимым символом.
   - Исправлена поддержка редактирования меток диапазона X страниц (celXMin / celXMax).

2. **Изменение размеров границ страницы (Resize)**:
   - При наведении курсора на левую/правую границы выбранной страницы курсор меняет форму на crSizeWE.
   - При наведении на верхнюю/нижнюю границы — на crSizeNS.
   - Нажатие левой кнопки мыши и перетаскивание изменяет соответствующее поле FloatRect страницы (нормализованные координаты пересчитываются относительно размеров компонента).

3. **Перетаскивание страницы (Move)**:
   - При наведении курсора на страницу с зажатой клавишей Ctrl курсор меняется на crSizeAll.
   - Зажатие левой кнопки мыши с Ctrl позволяет таскать страницу по области экрана (сдвигаются все координаты FloatRect).

3. **Ограничение вывода текста и осей границами страницы (GL_SCISSOR_TEST)**:
   - Ранее GL_SCISSOR_TEST включался только на время отрисовки серий графиков (RenderObject). Метки осей и сетка рисовались без теста отсечения, из-за чего при уменьшении размеров страницы подписи X-оси вылезали за правую границу страницы и накладывались на соседние страницы.
   - Теперь GL_SCISSOR_TEST включается на всё время рендеринга страницы (RenderPage), ограничивая отрисовку рамки, сетки и подписей осей прямоугольником PageRect страницы. Для графиков внутри страницы применяется вложенный, более строгий glScissor по области lContentRect.

4. **Примагничивание страниц к соседям и границам графика (Snapping/Magnetism)**:
   - Добавлено свойство SnapSensitivity: Integer (по умолчанию 5 пикселей) в TChartPanZoomListener.
   - Внутри MouseMove реализованы вспомогательные функции SnapX и SnapY, которые ищут ближайшие границы (левую, правую, верхнюю, нижнюю) других страниц и границы области выравнивания PageArea самого чарта в пределах заданной чувствительности.
   - **При ресайзе**: Перетаскиваемая граница автоматически прилипает к близким границам соседей или краям холста.
   - **При перемещении с Ctrl**: Вся страница целиком прилипает своими границами к соседним страницам или краям холста.

5. **Выделение региона (зумирование) и сброс масштаба (Ctrl + Drag)**:
   - **Условия запуска**: Если зажать клавишу Ctrl, нажать левую кнопку мыши внутри области построения (Plot Area / ContentRect) страницы.
   - **MouseMove**: Рассчитывается и рисуется полупрозрачная рамка выделения. Границы рамки автоматически ограничиваются областью построения.
   - **MouseUp**:
     - Если направление перемещения было **вправо и вниз** (координаты X и Y увеличились): происходит масштабирование (зум) выбранной области. Пиксели переводятся в значения осей через PixelToXValue и PixelToAxisValue для каждого элемента.
     - Если направление перемещения было **влево и вверх** (координаты X и Y уменьшились): происходит сброс масштаба к полному отображению всех данных (через рекурсивный обход дочерних элементов и вычисление глобальных границ min/max).
     - Рамка скрывается.
   - **Отрисовка рамки**: Добавлена поддержка в класс TOpenGLChartRenderer, которая рисует полупрозрачный синий прямоугольник с синей рамкой ($403C7DFF / $FF3C7DFF).
