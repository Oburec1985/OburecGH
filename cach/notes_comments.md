# Заметки по документированию модуля uOglChartFrameListener.pas

## Выполненные задачи
1. **Интерактивный курсор для осей**:
   - При наведении курсора на физическую ось графика курсор мыши меняется на `crHandPoint`, визуально подсказывая пользователю о возможности взаимодействия (клике или перетаскивании).
   - Логика встроена в метод `TChartSelectListener.MouseMove` с использованием хит-тестинга `GetAxisHitAt` рендерера `TOpenGLChartRenderer`.
   - При уходе курсора с оси форма возвращается в `crDefault`.

2. **Русские XML-комментарии и аннотации**:
   - Написаны полноценные русские комментарии в стиле XML-документации (`/// <summary>`, `/// <param>`) для классов:
     - `TChartFrameListener` (базовый слушатель)
     - `TChartSelectListener` (управление выделением и ховером)
     - `TChartPanZoomListener` (управление зумом, перемещением и размерами)
   - Детально проаннотирована внутренняя логика обработки мыши и клавиатуры:
     - Hit-testing текстовых элементов и осей.
     - Манипуляции с вершинами сплайнов Безье (добавление через Insert, удаление, выделение, смена типа вершин).
     - Попиксельное изменение размеров областей графиков за рамки границ и их перетаскивание с примагничиванием (SnapX / SnapY).
     - Обычный скролл-зум (`MouseWheel`) и зум по рамке (`MouseUp` / `MouseDown` с Ctrl + Drag).
     - Алгоритм автоматического подбора диапазона осей (`FitPageZoom`).

3. **Сохранение кодировки**:
   - Исходный файл сохранен строго в кодировке `Windows-1251` (cp1251) без `BOM` с переводами строк `CRLF`.
   - Все кириллические комментарии сохранены в корректном и читаемом виде.
   - Любые промежуточные файлы `.utf8` удалены.
