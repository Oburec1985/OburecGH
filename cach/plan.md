# План доработок - Задание 3 (Модуль программ ЦАП)

## Статус: Выполнено

## Архитектура:

### Программы (uDACProgram.pas):
- **TDacProgram** - базовый класс
  - Знает про устройство (fDevice: TDacDevice)
  - Сам подписывается на OnGenerateData
  - Сам управляет буферами (получает из очереди, заполняет, ставит обратно)
  - Методы: Start, Stop, SetDevice

- **TSimpleSinusProgram** - простая генерация синуса
  - Не требует изменения размера буфера
  
- **TAccurateSinusProgram** - умная генерация с кратностью периодов
  - При изменении частоты рассчитывает требуемый размер буфера
  - Автоматически вызывает ReallocateBuffers устройства

### Форма (Unit1.pas):
- ТОЛЬКО переключает программы
- ТОЛЬКО задает параметры (Frequency, Amplitude, MinSamplesTarget)
- НЕ содержит кода генерации данных

## Поток работы:

```
1. FormCreate:
   - Создает программы
   - Устанавливает Device для программ
   
2. btnPlayStopClick (Play):
   - Device.Open
   - Device.Start
   - Program.Start (подписывается на OnGenerateData)
   
3. Устройство вызывает OnGenerateData:
   - Программа получает буфер из очереди
   - Заполняет данными
   - Ставит буфер обратно в очередь
   
4. btnPlayStopClick (Stop):
   - Program.Stop (отписывается)
   - Device.Stop
```

## Ключевые изменения:
1. Программы знают про устройство
2. Программы сами управляют подпиской на события
3. Форма не содержит кода генерации
4. ReallocateBuffers вызывается программой при необходимости
