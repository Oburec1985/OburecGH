# План работ: Доработки OpenGL Chart и документирование

## Текущая задача
Добавление интерактивного изменения курсора мыши при наведении на оси графиков, а также детальное документирование и комментирование модуля `uOglChartFrameListener.pas` на русском языке в кодировке Windows-1251.

## Шаги
- [x] Доработка `MouseMove` в `TChartSelectListener` для изменения курсора мыши на `crHandPoint` при наведении на физические оси графиков.
- [x] Полное документирование XML-style для классов `TChartFrameListener`, `TChartSelectListener`, `TChartPanZoomListener` в секциях interface и implementation.
- [x] Подробное комментирование логики наведения мыши, выделения, перетаскивания вершин Безье, масштабирования (зума) по рамке и панорамирования в implementation секции.
- [x] Проверка кодировки Windows-1251 (cp1251) для сохранения русских комментариев без повреждений.
- [x] Удаление временных UTF-8 файлов для поддержания чистоты проекта.
